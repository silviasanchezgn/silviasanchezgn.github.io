Guía paso a paso para añadir un nuevo proyecto:

**1. Preparar la imagen**

Guarda la foto que quieras usar dentro de la carpeta llamada "Fotos". Asegúrate de recordar el nombre del archivo (ejemplo: mi-proyecto.jpg).

**2. Copiar la estructura base**

Abre el archivo sociales.html y busca el bloque de código entre las líneas 230 y 239.

Selecciona todo ese bloque (el contenedor div).

Cópialo (Ctrl + C o Cmd + C).

**3. Pegar en la ubicación correcta**

Desplázate hasta el final del archivo. Debes pegar el código justo en el hueco que hay:

Después del último /div.

Antes de la etiqueta script.

**4. Personalizar el contenido**

En el fragmento que acabas de pegar, localiza y cambia estos tres elementos para que correspondan a tu nuevo proyecto:

  Imagen: Cambia la ruta en src="..." por la de tu nueva foto (ej. Fotos/mi-proyecto.jpg).
  
  Título: Sustituye el texto del encabezado (entre etiquetas h3 ).
  
  Descripción: Cambia el texto del párrafo (entre etiquetas p ).
  
  Ubicación: Sustituye la ubicación con la que sea correcta para tu proyecto (entre etiquetas h3 ) (para seguir un mismo estilo utiliza el emoji       "📍" justo antes)


**💻 Nuestra Vitrina Digital: Proyectos La Salle**

¿Qué hay en la web?
Página Principal: Una bienvenida sencilla con la estrella de La Salle y un menú para moverte por el sitio sin perderte.

Proyectos por Categorías: Hemos dividido los trabajos por secciones. Cada uno tiene una foto grande, titulo, un resumen de lo que hicimos y el nombre del colegio que lo ha realizado.

Contacto: Pusimos una página con los datos de los dos colegios que participamos, por si alguien necesita escribirnos o saber dónde estamos.

¿Cómo lo hemos hecho?
Para que la web quedara así, hemos trabajado con varias herramientas:

  Usamos código HTML y CSS para montar la estructura y que todo estuviera en su sitio.
  
  Diseñamos el fondo y la parte visual con Canva para que la web tuviera nuestro propio estilo.
